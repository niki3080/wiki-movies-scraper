# Wikipedia Movies Scraper

Scrapy-паук для сбора информации о фильмах из русской Википедии 
(категория «Фильмы по алфавиту»).

Источник данных:  
https://ru.wikipedia.org/wiki/Категория:Фильмы_по_алфавиту

В итоговый CSV-файл (movies.csv) сохранено 37 480 записей о фильмах.

## Что собирает

- название фильма (из заголовка страницы `h1`)

Из блока infobox:

- жанр
- режиссёр
- страна
- год выпуска (только год)

---

## Технологии

- Python
- Scrapy
- XPath
- регулярные выражения

---

## Примечания

- текст очищается от сносок, лишних пробелов и артефактов разметки
- год выпуска сохраняется без даты и месяца
- если информация отсутствует на странице фильма, поле остаётся пустым
- переход по страницам категории реализован через ссылку "Следующая страница"

---

## Как запустить

Из корневой директории проекта:
S
```bash
scrapy crawl movies -o movies.csv
```
